# Veřejně dostupným QR kódům se nedá vždy věřit
Na tuto stránku jste se pravděpodobně dostali po naskenování veřejně umístěného QR kódu, který byl nahrazen odkazem sem.

Tentokrát je to neškodná ukázka. Útočník ale může stejným způsobem přesměrovat na podvržený web, jehož cílem je vylákat přihlašovací údaje, platební informace nebo jiné citlivé údaje.
